# -Introduction
This repository provides datasets for **fire protection facility detection** and **building material classification**.

The fire protection facility dataset is suitable for object detection tasks and demonstrates stable performance across multiple detection models, including 
YOLOv3n, YOLOv5n, YOLOv8n, YOLOv11n, SSD, Faster R-CNN, and RetinaNet.

Dataset 
**FF-Data** is a fire protection facility dataset consisting of images collected from real-world environments. It covers eight categories commonly found in practical scenarios, including:
- Fire extinguisher  
- Sprinkler   
- Fire door  
- Detector  
- Fire alarm   
- Fire hydrant  
- Emergency lighting  
- Exit   

**Material-Data** is a building material dataset designed for classification tasks. It includes four major categories of construction materials that cover most primary structural components in existing buildings:
- Brick and Mortar  
- Concrete  
- Steel  
- Timber
